# Kaun Banega Crorepati (KBC) - C++ Console Game

A text-based replica of the famous quiz show "Who Wants to Be a Millionaire" built entirely using Object-Oriented C++.

## 🚀 Features
- 5 progressively challenging general knowledge and programming questions.
- Realistic prize money scaling.
- Interactive **50:50 Lifeline** functionality.
- Automatic terminal screen clearing for a clean user interface.

## 🛠️ How to Compile and Run
1. Open your terminal.
2. Compile the code: `g++ kbc.cpp -o kbc_game`
3. Run the executable: `./kbc_game`
